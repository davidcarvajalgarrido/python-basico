# Lección 2 — Ejercicios

## Ejercicio 1: Variables y tipos básicos

1. Crea tres variables: `nombre` (string), `edad` (int) y `altura` (float).
2. Muestra por pantalla el tipo de cada una usando `type()`.
3. Concatena `nombre` y `edad` en un mensaje como: "Juan tiene 25 años".

## Ejercicio 2: Operadores aritméticos

1. Crea dos variables `a = 17` y `b = 5`.
2. Calcula y muestra: suma, resta, multiplicación, división, división entera y módulo.
3. Calcula `a` elevado a `b` usando el operador `**`.

## Ejercicio 3: Trabajando con strings

1. Dada la cadena `texto = "  PYTHON es Genial  "`, normalízala a minúsculas y sin espacios a los lados.
2. Reemplaza "genial" por "increíble".
3. Divide el string por espacios y muestra la lista resultante.

## Ejercicio 4: Listas - operaciones básicas

1. Crea una lista `frutas = ["manzana", "pera", "plátano"]`.
2. Añade "naranja" al final.
3. Inserta "fresa" en la segunda posición (índice 1).
4. Elimina "pera" de la lista.
5. Muestra el tercer elemento y la longitud total de la lista.

## Ejercicio 5: Diccionarios - gestión de datos

1. Crea un diccionario `alumno` con claves: `nombre`, `edad`, `curso` y valores apropiados.
2. Añade una nueva clave `nota` con valor 8.5.
3. Modifica el valor de `edad` incrementándolo en 1.
4. Muestra todas las claves del diccionario.
5. Comprueba si la clave `email` existe en el diccionario.

## Ejercicio 6: Tuplas - datos inmutables

1. Crea una tupla `coordenadas = (10.5, 20.3, 5.8)` que represente x, y, z.
2. Accede al segundo elemento (y) y muéstralo.
3. Intenta modificar el primer valor (debería dar error, comenta la línea).
4. Desempaqueta la tupla en tres variables separadas: `x, y, z`.

## Ejercicio 7: Conjuntos - elementos únicos

1. Crea dos conjuntos: `a = {1, 2, 3, 4, 5}` y `b = {4, 5, 6, 7, 8}`.
2. Calcula y muestra la unión de ambos conjuntos.
3. Calcula y muestra la intersección (elementos comunes).
4. Calcula la diferencia `a - b` (elementos en a pero no en b).
5. Añade el número 9 al conjunto `a` y elimina el 1.

## Ejercicio 8: Conversión entre tipos

1. Convierte la lista `[1, 2, 2, 3, 3, 3]` en un conjunto para obtener valores únicos.
2. Convierte el conjunto resultante de nuevo a lista ordenada.
3. Crea un diccionario a partir de dos listas: `claves = ["a", "b", "c"]` y `valores = [1, 2, 3]` usando `zip()`.

## Ejercicio 9: Operadores de comparación y lógicos

1. Crea dos variables `x = 10` y `y = 20`.
2. Comprueba si `x` es mayor que `y`.
3. Comprueba si `x` está entre 5 y 15 usando operadores lógicos (`and`).
4. Comprueba si `y` es igual a 20 o `x` es menor que 5 (usando `or`).

## Ejercicio 10: Práctica integradora

1. Crea una lista de diccionarios representando 3 productos: cada uno con `nombre`, `precio` y `stock`.
2. Añade un cuarto producto.
3. Crea una lista solo con los nombres de todos los productos.
4. Calcula el precio total sumando todos los precios (usa un bucle `for`).
