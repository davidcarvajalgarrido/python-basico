# Lección 2 — Ejercicios

## Ejercicio 1: Cálculos y redondeo

1. Calcula el área de un círculo de radio 3.1415 usando `math.pi` y muestra el resultado con 3 decimales.
2. Convierte el resultado a cadena con f-string y añade la unidad `u²`.

## Ejercicio 2: Normalización de texto

1. Dada la cadena `'  PyTHon,básico , CURSO '`, normalízala a `'python,básico,curso'` sin espacios extra.
2. Separa por comas y crea una lista de términos.

## Ejercicio 3: Diccionario de conteos

1. Con la lista `['rojo','azul','rojo','verde','azul','rojo']`, construye un diccionario con las frecuencias de cada color.
