# Lección 4 — Ejercicios

## Ejercicio 1: Normalizador con parámetros

1. Escribe `normaliza(texto, *, lower=True, trim=True)` que aplique `strip()` y/o `lower()` según flags.
2. Prueba con `'  Hola PYTHON  '` usando distintas combinaciones.

## Ejercicio 2: Orden personalizado

1. Ordena una lista de tuplas `(nombre, edad)` por edad ascendente y, a igual edad, por nombre insensible a mayúsculas.

## Ejercicio 3: Factorial con validación

1. Implementa `factorial(n)` con retorno correcto para `n>=0` y lanza `ValueError` si `n` es negativo.
