# Lección 3 — Ejercicios

## Ejercicio 1: Validación de contraseñas

1. Pide una contraseña por input y valida que tenga al menos 8 caracteres y contenga un dígito.
2. Imprime mensajes claros para cada caso y repite hasta que sea válida.

## Ejercicio 2: Suma de pares

1. Suma los números pares del 1 al 100 usando un `for` y comprueba el resultado esperado (2550).

## Ejercicio 3: Búsqueda con else en bucles

1. Dada una lista de nombres, busca uno objetivo. Si no aparece, usa el `else` del bucle para indicar que no se encontró.
